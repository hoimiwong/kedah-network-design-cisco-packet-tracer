# Kedah Network Design (Cisco Packet Tracer)

## Description
Designed and implemented a network topology for the Kedah branch as part of a multi-branch enterprise network system using Cisco Packet Tracer.

## My Contribution (Kedah Branch)
- Designed logical topology using star topology for multiple departments  
- Configured network devices including routers, switches, and access points  
- Planned and implemented IP addressing scheme using VLSM  
- Designed secure wireless network using WPA2-PSK authentication  
- Ensured communication between departments and across branches  

## Key Features
- Star topology for improved reliability and scalability  
- Department-based network segmentation (IT, Office, Finance, HR)  
- Router-based communication between departments and branches  
- Wireless connectivity for IoT and mobile devices  
- Efficient IP allocation using VLSM  

## Technologies Used
- Cisco Packet Tracer  
- Networking Concepts (LAN, WAN, VLSM)  
- Router and Switch Configuration  
- Wireless Network Configuration (WPA2-PSK)

This project demonstrates practical understanding of network design, IP addressing, and configuration of network devices in a real-world enterprise environment.
